# Lucidaeon Tap

Homebrew formulae for lucidaeon's command-line tools, bottled for macOS.

## Formulae

| formula | what it does |
|---|---|
| [starcat](https://github.com/lucidaeon/mediumcoeli) | ephemeris computation + table presentation |
| [blackmoon](https://github.com/lucidaeon/mediumcoeli) | reads / merges / writes astrology chart formats |

## Install

```sh
brew install lucidaeon/tap/starcat
brew install lucidaeon/tap/blackmoon
```

Or `brew tap lucidaeon/tap` and then `brew install starcat` / `brew install blackmoon`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "lucidaeon/tap"
brew "starcat"
brew "blackmoon"
```
