# Lucidaeon Tap

## How do I install these formulae?

`brew install lucidaeon/tap/<formula>`

Or `brew tap lucidaeon/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "lucidaeon/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
